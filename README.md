# What is SpyenV2?

**SpyenV2** is a complete rewrite of the original [**Spyen**](https://github.com/Buzau16/Spyen) â€” a lightweight, code-only **game engine** built for developers who prefer full control and zero bloat.  
This version focuses on speed, modularity, and clean design, offering a solid foundation for building 2D (and potentially 3D) games entirely through code.

---

##  Installation

Clone the repository:

```bash
git clone https://github.com/Buzau16/SpyenV2.git
cd SpyenV2/
```
Run the build.bat script

Open the generated solution in Visual Studio 2024
